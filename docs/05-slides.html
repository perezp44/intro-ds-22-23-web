Transparencias/Slides: en esta sección iré subiendo las transparencias de clase conforme las vaya preparando. Las primeras transparencias están hechas en el formato ioslides. Para el resto uso el formato xaringan.

Todas las transparencias están hechas con R, de hecho, una de las cosas que has de aprender durante el curso es a hacer transparencias desde R. Concretamente, habrá un trabajo en equipo (3 personas), donde entre otras cosas, tendréis que hacer unas transparencias para presentar vuestro proyecto. Tendréis que hacer las transparencias con R, usando el formato ioslides o quizás el nuevo formato revealjs.
